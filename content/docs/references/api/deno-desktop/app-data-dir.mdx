The OS application-data base directory where downloaded bundles persist:
macOS `~/Library/Application Support`, Windows `%APPDATA%`, Linux `$XDG_DATA_HOME` (or
`~/.local/share`). Override with the `WVB_APP_DATA_DIR` env var.
