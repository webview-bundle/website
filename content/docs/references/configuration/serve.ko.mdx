---
title: serve
description: 'wvb.config의 serve 섹션 — 미리보기 서버의 로컬 파일, 포트, 로그 출력입니다.'
---

`serve`(`ServeConfig`)는 [`wvb serve`](/docs/references/cli/serve)의 기본값을 설정합니다. `wvb
serve`는 개발용으로 패킹된 번들을 제공하는 로컬 서버입니다.

```ts title="wvb.config.ts"
import { defineConfig } from '@wvb/config';

export default defineConfig({
  serve: {
    file: './.wvb/app.wvb',
    port: 4312,
    silent: false,
  },
});
```

| Field    | Type      | Default                 |
| -------- | --------- | ----------------------- |
| `file`   | `string`  | the `pack.outFile` path |
| `port`   | `number`  | `4312`                  |
| `silent` | `boolean` | —                       |

`file`을 생략하면 해석된 `pack.outFile` 경로로 대체됩니다. 요청 로그 출력을 끄려면 `silent`를
설정하세요.

<Callout type="info">
  `serve`에는 `hostname` 필드가 없습니다. 다른 호스트에 바인딩하려면 명령줄에서 `wvb serve`에
  `--hostname`(별칭 `-H`)을 전달하세요.
</Callout>
