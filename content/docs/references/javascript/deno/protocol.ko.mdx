---

Deno 바인딩은 두 가지 프로토콜 핸들러를 제공합니다. `BundleProtocol`은 커스텀 스킴을 통해 번들의 파일을 웹뷰에 제공합니다. `LocalProtocol`은 개발용으로 `app://host/...` URL을 로컬호스트 주소로 프록시합니다. 두 핸들러 모두 요청을 FFI 응답으로 해석하며, `toResponse(res)`로 표준 `Response`로 변환합니다.

기본 개념은 [프로토콜 처리](/docs/guide/core-concepts/protocol-handling)를 참고하세요. Deno가 제공하는 전체 API와 네이티브 라이브러리를 로드하는 방법은 [Deno API 개요](/docs/references/javascript/deno)를 참고하세요.

## 핸들러

`BundleProtocol`은 로드된 `lib`과 [`BundleSource`](/docs/references/javascript/deno/source)를 받습니다. 들어오는 각 요청을 해석된 번들 안의 파일로 매핑합니다.

`LocalProtocol`은 `app://host/...` 요청을 로컬호스트 주소로 프록시합니다. 개발 서버를 대상으로 작업할 때 유용합니다.

각 핸들러는 HTTP 메서드, 요청 URL, 헤더 객체를 받아 FFI 응답으로 해석하는 `handle` 메서드를 제공합니다. 이 응답은 `toResponse(res)`로 웹 `Response`로 변환합니다.

| Member                         | Description                                                                                                                |
| ------------------------------ | -------------------------------------------------------------------------------------------------------------------------- |
| `handle(method, url, headers)` | 요청을 FFI 응답으로 해석합니다. `method`는 `HttpMethod`, `url`은 요청 URL 문자열, `headers`는 요청 헤더의 일반 객체입니다. |
| `toResponse(res)`              | FFI 응답을 표준 `Response`로 변환합니다. 독립 함수로 내보내집니다.                                                         |
| `free()`                       | 네이티브 핸들을 해제합니다. `using` / `[Symbol.dispose]`로도 사용할 수 있습니다.                                           |

`HttpMethod`는 허용되는 요청 메서드의 유니온 타입입니다. 예를 들어 `'get'`과 `'head'`가 있습니다.

## 번들 제공하기

`BundleSource`를 생성하고 `BundleProtocol`에 전달한 뒤, `Deno.serve` 콜백에서 `handle`을 호출하세요. 결과를 반환하기 전에 `toResponse`로 변환하세요.

```ts title="serve.ts"
import { loadLibViaPlug, BundleProtocol, BundleSource, toResponse } from 'jsr:@wvb/deno';

const lib = await loadLibViaPlug();

const source = new BundleSource(lib, {
  builtinDir: './bundles/builtin',
  remoteDir: './bundles/remote',
});

using protocol = new BundleProtocol(lib, source);

Deno.serve(async req => {
  const res = await protocol.handle('get', req.url, Object.fromEntries(req.headers));
  return toResponse(res);
});
```

## 핸들러 해제하기

프로토콜 클래스는 네이티브 핸들을 소유하므로 `Disposable`입니다. `using`으로 바인딩하면 스코프를 벗어날 때 해제되며, 작업이 끝나면 `free()`를 호출해 해제할 수도 있습니다.

```ts
using protocol = new BundleProtocol(lib, source);
// 스코프 끝에서 자동으로 해제됨
```

<Callout type="info">
  `BundleProtocol`은 [`BundleSource`](/docs/references/javascript/deno/source)에서 읽어옵니다.
  [Remote](/docs/references/javascript/deno/remote),
  [Updater](/docs/references/javascript/deno/updater)와 함께 사용하면 더 새로운 번들을 무선(OTA)으로
  가져와 설치할 수 있습니다.
</Callout>
