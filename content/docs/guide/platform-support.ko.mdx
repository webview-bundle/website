하나의 공유 Rust 코어가 모든 통합을 구동합니다. 따라서 한 번 패킹한 `.wvb`는 Electron, Tauri, Android, iOS, Deno Desktop에서 동일하게 로드됩니다. 코어가 각 호스트에 도달하는 방식과 웹뷰가 브리지를 통해 응답하는 방식은 [플랫폼 통합](/docs/guide/platform-integration)을 참고하세요.

이 페이지는 지원 매트릭스입니다. 플랫폼별 패키지 또는 크레이트, 최소 호스트 버전, 상태를 정리합니다.

## 지원 매트릭스

| 플랫폼                                       | 웹뷰 호스트    | 패키지 / 크레이트                 | 최소 버전                                                                        | 상태   |
| -------------------------------------------- | -------------- | --------------------------------- | -------------------------------------------------------------------------------- | ------ |
| [Electron](/docs/guide/platforms/electron)   | Chromium       | `@wvb/electron` 0.1.0 (npm)       | `electron >= 15`                                                                 | 안정   |
| [Tauri desktop](/docs/guide/platforms/tauri) | System WebView | `wvb-tauri` 0.1.0 (crate)         | Tauri v2                                                                         | 안정   |
| Tauri mobile                                 | System WebView | `wvb-tauri` 0.1.0 (crate)         | [Android](/docs/guide/platforms/android) / [iOS](/docs/guide/platforms/ios) 참고 | 안정   |
| [Android](/docs/guide/platforms/android)     | System WebView | `webview-bundle-android` (Kotlin) | minSdk 24 / Android 7.0                                                          | 안정   |
| [iOS](/docs/guide/platforms/ios)             | WKWebView      | `webview-bundle-ios` (Swift)      | iOS 16 / macOS 12                                                                | 안정   |
| [Deno Desktop](/docs/guide/platforms/deno)   | Deno webview   | `@wvb/deno-desktop` (JSR)         | —                                                                                | 실험적 |

## 데스크톱

Electron과 Tauri 데스크톱은 가장 성숙한 통합입니다. 둘 다 배포되어 있으며, Rust 코어를 기반으로 호출자가 제공한 URL 스킴을 통해 번들을 제공합니다.

### Electron

npm 패키지를 설치하세요.

```sh
npm install @wvb/electron
```

프로토콜을 등록하고 이름으로 번들을 로드하세요(`<scheme>://<bundleName>.wvb`).

```ts title="main.ts"
import { app, BrowserWindow } from 'electron';
import { bundleProtocol, wvb } from '@wvb/electron';

const instance = wvb({
  protocols: [bundleProtocol('app', { onError: e => console.error('[wvb]', e) })],
});

app.whenReady().then(async () => {
  await instance.whenProtocolRegistered();
  const window = new BrowserWindow({
    webPreferences: { contextIsolation: true, nodeIntegration: false },
  });
  await window.loadURL('app://hacker-news.wvb');
});
```

[Electron 가이드](/docs/guide/platforms/electron)에서 시작하세요.

### Tauri

Tauri 통합은 **Tauri v2**를 대상으로 하는 crates.io의 `wvb-tauri` Rust 크레이트입니다. `@wvb/tauri` npm 패키지는 없습니다. 동일한 크레이트가 Tauri의 모바일 타깃도 구동합니다.

```toml title="src-tauri/Cargo.toml"
[dependencies]
wvb-tauri = "0.1.0"
```

```rust title="src-tauri/src/lib.rs"
use wvb_tauri::{Config, Protocol, Source};

tauri::Builder::default()
  .plugin(wvb_tauri::init(
    Config::new()
      .source(Source::new())
      .protocol(Protocol::bundle("bundle")),
  ))
  .run(tauri::generate_context!())
  .expect("error while running tauri application");
```

그러면 윈도우가 `bundle://hacker-news.wvb`를 로드합니다. [Tauri 가이드](/docs/guide/platforms/tauri)에서 시작하세요.

## 모바일

Android와 iOS는 실제로 동작하는 통합입니다.

### Android

`webview-bundle-android` Kotlin 라이브러리(네임스페이스 `dev.wvb`)는 **minSdk 24(Android 7.0)**를 요구하며, `WebViewClient`를 통해 번들을 제공합니다. 네이티브 바인딩을 받은 다음 `WebView`에 설치하세요.

```sh
node scripts/install.mjs
```

```kotlin title="MainActivity.kt"
val wvb = WebViewBundle.getInstance(
  this,
  WebViewBundleConfig(protocols = listOf(WebViewBundleProtocol.bundle())),
)
val webView = WebView(this)
webView.settings.javaScriptEnabled = true
wvb.install(webView)
webView.loadUrl("https://app.wvb/")
```

번들은 일반적인 `https://<bundle>.wvb/...` URL로 제공됩니다(번들 이름이 첫 번째 호스트 레이블입니다). [Android 가이드](/docs/guide/platforms/android)를 참고하세요.

### iOS

`webview-bundle-ios` Swift 패키지는 **iOS 16**과 **macOS 12**를 대상으로 하며, `WKURLSchemeHandler`를 통해 번들을 제공합니다. 네이티브 xcframework 릴리스를 설치한 다음 `WKWebViewConfiguration`에 커스텀 스킴을 등록하세요.

```sh
node scripts/install.mjs ffi/0.1.0
```

```swift title="ContentView.swift"
let instance = try WebViewBundle.configure(
  WebViewBundleConfig(protocols: [.bundle(scheme: "app")])
)
let config = WKWebViewConfiguration()
instance.install(on: config)
let webView = WKWebView(frame: .zero, configuration: config)
webView.load(URLRequest(url: URL(string: "app://app.wvb")!))
```

[iOS 가이드](/docs/guide/platforms/ios)를 참고하세요.

## Deno Desktop

Deno Desktop은 가장 최신 통합이며, JSR에 `@wvb/deno-desktop`으로 올라와 있습니다. 번들 소스를 빌드하고 `Deno.serve` 호환 핸들러를 노출하며, 윈도우마다 하나의 프로토콜을 사용합니다.

<Callout type="info">
  Deno Desktop은 실험적입니다. [Deno Desktop 가이드](/docs/guide/platforms/deno)와 [Deno API
  레퍼런스](/docs/references/deno)를 참고하세요.
</Callout>
