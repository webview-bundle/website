## 브릿지 호출하기

(브릿지 호출시 네이티브 단에 등록된 핸들러가 브릿지 호출을 처리해줌. 각 플랫폼/프레임워크 별 가이드)

(Electron: ipc / Tauri : ipc / Android : JavaScriptInterface / iOS : WKScriptMessageHandler / Deno Desktop : Bindings(CEF))
