title: Start with an example
---

Use `create-wvb` to scaffold an example project with your preferred tool.

<Tabs items={['npm', 'pnpm', 'yarn']}>
<Tab value="npm">

```sh
npx create-wvb@latest
```

  </Tab>
  <Tab value="pnpm">
```sh
pnpx create-wvb@latest
```
  </Tab>
  <Tab value="yarn">
```sh
yarn dlx create-wvb@latest
```
  </Tab>
</Tabs>
