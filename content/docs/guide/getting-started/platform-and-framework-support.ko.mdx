웹뷰 번들은 여러 네이티브 플랫폼/프레임워크를 지원합니다.

## 지원 현황

| 플랫폼       | 호환 버전                     | 상태   |
| ------------ | ----------------------------- | ------ |
| Electron     | Electron >= 15                | 안정적 |
| Tauri        | Tauri v2 / Windows >= 10      | 안정적 |
| Android      | Android minSdk >= 24          | 안정적 |
| iOS          | iOS 16 / macOS 12.0 / Swift 6 | 안정적 |
| Deno Desktop | Deno >= 2.9.0                 | 실험적 |

더 많은 플랫폼/프레임워크 지원이 필요하다면 [깃헙 논의](...)에 요청사항을 적어주시면 감사하겠습니다.
