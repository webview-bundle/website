`create-wvb` 를 사용하여 선호하는 도구로 예제 프로젝트를 시작해 보세요.

<Tabs items={['npm', 'pnpm', 'yarn']}>
<Tab value="npm">

```sh
npx create-wvb@latest
```

  </Tab>
  <Tab value="pnpm">
```sh
pnpx create-wvb@latest
```
  </Tab>
  <Tab value="yarn">
```sh
yarn dlx create-wvb@latest
```
  </Tab>
</Tabs>
