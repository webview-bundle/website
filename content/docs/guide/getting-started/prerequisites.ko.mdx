---
title: 필요 사항
---

프로젝트를 시작하려면 먼저 몇 가지 종속성을 설치해야 합니다.

## Node.js

웹뷰 번들 커맨드라인 사용을 위해 [Node.js](https://nodejs.org/ko)를 설치해주세요.

- 최소 요구 버전 : 18
- 권장 버전 : LTS

## Android

안드로이드 앱 개발을 위해서 Android SDK 설치가 필요합니다. (혹은 [Android Studio](https://developer.android.com/studio)를 설치)

사용 도구의 최소 요구 버전은 다음과 같습니다.

- JDK 17
- Android 최소 SDK 버전 : 24

## iOS

iOS 앱 개발을 위해서 [Xcode](https://developer.apple.com/xcode/) 설치가 필요합니다.

사용 도구의 최소 요구 버전은 다음과 같습니다.

- Swift v5
- iOS 16

## Deno

[Deno Desktop](/docs/guide/native/deno)을 사용한다면 Deno 설치가 필요합니다.

---

플랫폼 별로 필요한 종속성은 각 프레임워크의 가이드를 참고해주세요.
