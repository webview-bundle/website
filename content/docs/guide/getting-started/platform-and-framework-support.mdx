---
title: Platform and framework support
description: The platforms and frameworks that run Webview Bundle.
---

Webview Bundle supports several native platforms and frameworks.

## Support status

| Platform     | Compatible versions           | Status       |
| ------------ | ----------------------------- | ------------ |
| Electron     | Electron >= 15                | Stable       |
| Tauri        | Tauri v2 / Windows >= 10      | Stable       |
| Android      | Android minSdk >= 24          | Stable       |
| iOS          | iOS 16 / macOS 12.0 / Swift 6 | Stable       |
| Deno Desktop | Deno >= 2.9.0                 | Experimental |

If you need support for more platforms or frameworks, please file a request in the [GitHub discussions](https://github.com/webview-bundle/webview-bundle/discussions).
