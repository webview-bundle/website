# 필요 사항

프로젝트를 시작하려면 먼저 몇 가지 종속성을 설치해야 합니다.

## Node.js

웹뷰 번들 커맨드라인 사용을 위해 [Node.js](https://nodejs.org/ko)를 설치해주세요.

- 최소 요구 버전 : 18
- 권장 버전 : LTS

## Android

안드로이드 앱 개발을 위해서 Android SDK

- JDK 버전
- Android SDK 지원 버전
- 동적 라이브러리 사용 중

## iOS

- Swift 버전
- macOS, iOS 지원 버전

## Deno

[Deno Desktop](...)을 사용한다면 Deno 설치가 필요합니다.

---

플랫폼 별로 필요한 종속성은 각 프레임워크의 가이드를 참고해주세요.
